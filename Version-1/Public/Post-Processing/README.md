Post-processing effects for MetaHuman scenes. Covers bloom, exposure, chromatic aberration, film grain, vignette, sharpening, color grading (global/shadows/midtones/highlights), color temperature, lens flare, film tone curve, and more.

Effects/ - Individual post-processing effect commands organized by category
Prest-Effects/ - Static color grading presets (batch-send all color grading commands at once)
Animated-Effects/ - Time-based animated post-processing effects that cycle values at 30fps
