Build Your Own World - POST

Core Spawn Commands:
SPWN_{tag}_{Category.Object}_{x}_{y}_{z}_{rotX}_{rotY}_{rotZ}_{scaleX}_{scaleY}_{scaleZ} - Spawn object
SPWNMOVE_{tag}_{x}_{y}_{z}_{rotX}_{rotY}_{rotZ}_{scaleX}_{scaleY}_{scaleZ} - Move/transform object
SPWNDEL_{tag} - Delete specific object by tag
SPWND - Delete ALL spawned objects

Tag Rules: Must be unique, no underscores (used as delimiters).

Special Object Types:
  Live.Plane - Background image/video plane (supports BACK_ and m3u8_ commands)
  LIGHT.Volume - Volumetric light (supports LIGHTC_ and LIGHTI_ commands)
  Procedural.Rock - Procedural rock with foliage (supports PROC* commands)
  Meta.* - Additional MetaHuman characters

Procedural Rock Commands (for Procedural.Rock objects):
  PROCM_{tag}_{meshName} - Set base mesh (ForestRock1-10)
  PROCG_{tag}_{meshName} - Set grass mesh (Grass1-3)
  PROCGD_{tag}_{value} - Grass density (0.01-1)
  PROCGZ_{tag}_{value} - Grass Z offset (0-1)
  PROCM_{tag}_Ferns_Mesh_{meshName} - Set ferns mesh (Fern1-2)
  PROCFD_{tag}_{value} - Ferns density (0-1)
  PROCFZ_{tag}_{value} - Ferns Z offset (0-1)
  PROCR_{tag}_{meshName} - Set rocks mesh (RiverRock1-3, RiverRockGroup)
  PROCRD_{tag}_{value} - Rocks density (0-1)
  PROCRZ_{tag}_{value} - Rocks Z offset (0-1)
  PROCP_{tag}_{meshName} - Set pebbles mesh (Pebbles1-2)
  PROCPD_{tag}_{value} - Pebbles density (0-1)
  PROCPZ_{tag}_{value} - Pebbles Z offset (0-1)
  PROCL_{tag}_{meshName} - Set logs mesh (Log1-6)
  PROCLD_{tag}_{value} - Logs density (0-1)
  PROCLZ_{tag}_{value} - Logs Z offset (0-1)
