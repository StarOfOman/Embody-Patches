Scene Presets
Pre-built environments loaded from Enviornments.txt. Each sends a batch of SPWN commands.

Loading Sequence:
  1. SPWND (clear all objects)
  2. DL_Int_0 (disable directional light)
  3. Wait 500ms
  4. Send each SPWN command with 10ms delay
