Eye customization for MetaHumans. 43 parameters across 3 sections: Pupil & Iris Shape, Iris Color, and Sclera. All use the EYE_{param}_{value} command format.

Commands/ - Individual eye slider commands (43 EYE_ float parameters)
Eyes-Presets/ - Static preset combinations (batch all EYE_ commands at once)
Animated-Eyes/ - Time-based animated eye effects that cycle values at 30fps
