Hair customization for MetaHumans. Supports hairstyle selection (HS_), base color (HCR/HCG/HCB), float sliders (HAIRF), color pickers (HAIRC), static presets, and animated hair effects.

Commands/ - Individual hair slider, color, and hairstyle commands
Hair-Presets/ - Static preset combinations (batch all HAIRF + HAIRC + HCR/HCG/HCB commands at once)
Animated-Hair/ - Time-based animated hair effects that cycle colors/values at 30fps
