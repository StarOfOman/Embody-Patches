Makeup customization for MetaHumans. Supports individual float sliders (MKUPF), color pickers (MKUPC), static presets, and animated makeup effects.

Commands/ - Individual makeup slider and color commands
Makeup-Presets/ - Static preset combinations (batch all MKUPF + MKUPC commands at once)
Animated-Makeup/ - Time-based animated makeup effects that cycle colors/values at 30fps
