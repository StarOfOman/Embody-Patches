# Voices - ElevenLabs

Voice IDs used for text-to-speech via the ElevenLabs API. Each voice must be initialized before use with the `ElevenLabs=` command, then spoken with `TTS_ElevenLabs_`.

## Initialization
```
ElevenLabs={apiKey}={voiceId}
```
